OpenDojo CMSclose

Taking on multiple roles in this project, my contributions involved server setup with Node and Express.js, implementing a rudimentary front-to-back authentication module using JSON Web Tokens, as well as being largely responsible for the style guide and appearance of the front-end view elements.

This project is a very classic case of open-source methodology done right - from the coding rules agreed upon by the entire team, to everyone equally contributing to the documentation. Most importantly, this was a very fast-paced learning experience that focused on everyone being out of their comfort zone and coping with brand new tech stacks and frameworks, with me personally confronting React.js and JWTs for the first time.
